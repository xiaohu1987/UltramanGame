# 奥特曼打怪兽 · 回合制小游戏

纯前端（HTML / CSS / JavaScript）回合制对战小游戏：  
玩家选择 **3 名奥特曼**，系统随机派出 **3 只 AI 怪兽**，双方轮流行动并释放技能，直到一方全灭。

## 如何游玩

### 方式一：本地静态服务器（推荐）

在项目根目录执行：

```bash
# Python 3
python -m http.server 8080
```

浏览器打开：`http://127.0.0.1:8080`

### 方式二：直接打开

双击 `index.html` 也可游玩；若图片路径含空格，建议使用本地静态服务器。

当前本地验证地址示例：`http://127.0.0.1:8765/`

## 操作说明

1. **选角阶段**：从候选奥特曼中点选恰好 3 名，点击「开始对战」。
2. **战斗阶段**：
   - 按速度决定行动顺序
   - 玩家回合：点击技能；若技能需要目标，再点击高亮角色
   - AI 回合：自动选择技能与目标
3. **技能类型**：
   - 伤害：攻击敌方
   - 治疗：恢复生命
   - 增益：提升攻击
   - 减益：降低防御
4. **结算**：一方全灭后弹出胜负结果，可「再来一局」。
